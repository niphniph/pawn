# Gambit Wallpapers - Premium Gaming Wallpaper Gallery & Puzzle Board

A high-fidelity, mobile-first wallpaper discovery app inspired by Wallz.gg but rebuilt from scratch. It features a custom gaming design language modeled after the provided game board visual: vibrant neon blue and pink bubbles, yellow-gold metallic barriers, and glossy 3D overlay highlights.

---

## 🚀 Features

1. **Vibrant 3D Glass/Glossy Styles**: Custom styling recreating the bubble-like blue/pink blocks, gold pegs, and dark cells from the game board.
2. **Explore & Discovery Grid**: Search wallpapers instantly by title or tags, and filter through responsive tags (Anime, Cyberpunk, 3D Glossy, Minimalist, etc.).
3. **Interactive Pawn Gambit Puzzle Board**: Fully interactive companion mini-game built inside the app recreating the exact board from the photo. Play by moving the pink and blue glossy balls past yellow barrier capsules to reach their respective finish zones.
4. **Interactive Details Modal**: Click any wallpaper to open a detailed layout showing its resolution, author, dimensions, and related tags, plus custom simulated download indicators and favorite saving.
5. **Favorites Sync**: Mark wallpapers you love and see them synchronized and saved inside your browser's local storage.
6. **Mobile-First Layout**: Sleek responsive design with a bottom tab-bar on mobile and a complete companion sidebar on desktop.

## 🛠️ Tech Stack

- **React** (v18+)
- **Vite** (Next-generation build tool)
- **TypeScript** (Strict static types)
- **Tailwind CSS** (v3.4 styling)
- **Lucide React** (Vector icons)

## 📦 Setup & Installation

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build Production Assets**:
   ```bash
   npm run build
   ```
